Course Template Converter

Applies the RDP style to your Brightspace course. Your wording, links, files and pictures stay exactly as they are — it only styles what's already there. Three steps, about five minutes.

Is this safe? Will it change my real course?

It's safe, and it won't. Your real Brightspace course is never touched — this only makes a new file from the copy you export, which you then choose whether to import.

Your course file is never uploaded anywhere. There is no server: everything happens inside your own browser on your own computer. You could disconnect from the internet and this page would still work.

Works best in an up-to-date Chrome or Edge.

Step 1  Get your course file out of Brightspace

You need the file ending in .imscc. Already have it? Go straight to Step 2.

Show me how to export it — 6 steps, with pictures
  1. Course Admin (in the course navbar, along the top)
  2. Import / Export / Copy Components
  3. Choose Export as Common Cartridge
    Not "Export as Brightspace Package", not "Thin Common Cartridge".
  4. Tick Select All Components, then Continue
  5. Check the summary, then click Export — every section should say "Export all items".
  6. Wait for "The course export was successful", then click "Click here to download the export Zip package" — the export takes a minute or two. Don't click Done until you've downloaded it.

That download is your course file — something like YourCourse.imscc. Take it to Step 2 exactly as it is: don't open, unzip, or re-zip it.

Brightspace calls it a "Zip package", so it's tempting to unzip it. Don't — this tool needs it whole, exactly as it downloaded.

If something looks different, or it went wrong

My screen doesn't match / I'm not sure which export to pick

Only plain Common Cartridge includes your actual page content. "Export as Brightspace Package" and "Thin Common Cartridge" will look like they worked and then produce a file this tool can't do anything useful with.

Which components should I tick?

Tick the top box, Select All Components — that gets everything (Content, Assignments, Quizzes, and so on). When in doubt, take it all: including extra components does no harm, and leaving things out is what usually goes wrong. If it asks whether to include all items or pick them individually, choose all items.

I clicked Done / navigated away, and never got the download

Nothing is lost — the export is still waiting for you. When it's ready, the bell (🔔) in the top bar picks up an orange dot:

Click the bell and you'll see Export finished. Open that to get back to the download link.

The export runs in the background, so it's also fine to close the tab and come back for it later.

I can't see ".imscc" on the end of the file name

That's normal — Windows usually hides it. As long as you chose Common Cartridge, it's the right file.

My Mac turned it into a folder instead of a file

Safari unzipped it for you. Turn off Safari → Settings → General → "Open safe files after downloading", then download it again. Don't try to re-zip the folder yourself — that makes a .zip, which is not the same thing.

This is a Blackboard course, not Brightspace

This tool works on Brightspace exports. A Blackboard export won't have the right shape inside.
