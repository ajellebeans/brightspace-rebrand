RDP Course Rebrand

This gives your Brightspace course the RDP look. Your wording, links, files and pictures stay exactly the same — only the design changes. Everything happens right here in your web browser: your course is never uploaded, and your real Brightspace course is never touched — this only makes a new file from the copy you export. Works best in an up-to-date Chrome or Edge.

Step 1  Get your course file out of Brightspace

You need the one file Brightspace makes when you export a course. In your course, go to:

  1. Course Admin (or the gear / settings menu)
  2. Import / Export / Copy Components
  3. Choose Export as Common Cartridge — not "Export as Brightspace Package" and not "Thin Common Cartridge". Only plain Common Cartridge includes your actual page content.
  4. Click Export and wait a moment while Brightspace builds it. When it's done, you'll get a notification on the bell (🔔) at the top of the page — click it, then download the file from there.

That download is your course file — it'll be named something like YourCourse.imscc. Can't see the ".imscc" part on the end? That's normal — Windows usually hides it. As long as you chose Common Cartridge above, it's the right file.

⚠️ Use the file exactly as Brightspace gave it — don't open, unzip, or re-zip it. (On a Mac, if Safari unzips it into a folder on its own, turn off Safari → Settings → General → "Open safe files after downloading," then download again.)

Step 3  Put it back into Brightspace

    
When it's done, you'll see one or two download buttons above:

    
    
      
  • REBRANDED — the new look, same pages as before (the safe choice)
    • 
      
  • REBRANDED (combined) — the new look, with very short pages merged together for less clicking
    • 
    

    
Import one back into Brightspace:
       Course Admin → Import / Export / Copy Components → Import Components → Start → Upload your file → Import All Components.

    
Always import into a practice / sandbox course first and click around —
       especially your quizzes, assignments, discussions and any release conditions, since those don't always
       carry across the Common Cartridge format perfectly (your pages, files, images and links are what this tool protects).
       No sandbox? Ask CTLS / LMS support for one. If the combined version looks off,
       use the plain REBRANDED file instead.

    
💡 Good to know: this tool never changes
       your real course — it only makes a new file. When you import into your live course, the import adds
       pages as new modules (it never overwrites), so delete the old modules first to avoid two copies.
       Your exported .imscc also makes a handy backup.
